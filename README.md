## üìñ Course Description
This course covers deep learning concepts, hyperparameters, and their optimization. Students will learn to apply deep learning algorithms using the **PyTorch** framework and the **Python** programming language to solve specific dataset problems.

### Capstone Learning Objectives (CPMK):
* **CPMK 1:** Explain deep learning concepts, hyperparameters, and the difference between traditional programming and AI.
* **CPMK 2:** Apply mathematical calculations for network weight updates (Backpropagation).
* **CPMK 3:** Analyze model training results and implement solutions using PyTorch.
